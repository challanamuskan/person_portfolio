## ✨ Features

- 🎬 Netflix-inspired dark UI with red accent theme

- 🤖 Built-in AI chatbot (Jerry) — rule-based, no API key needed

- 📊 Hero stats bar — live metrics visible on load
- 🔗 Direct links to live projects and GitHub repos

## 🚀 Run Locally
```bash
git clone https://github.com/challanamuskan/person_portfolio
cd person_portfolio
npm install
npm run dev
```

Open http://localhost:5173

## 📦 Build for Production
```bash
npm run build
npm run preview
```
